Politica de Privacidade

Ultima atualizacao: 1 de marco de 2026

1. Dados Coletados

Coletamos apenas os dados que voce fornece voluntariamente:

  • Nome, e-mail e WhatsApp (ao usar nossas ferramentas)
  • Respostas de quizzes e formularios
  • Dados de navegacao anonimizados (analytics)

2. Uso dos Dados

Seus dados sao usados para:

  • Entregar os resultados das ferramentas
  • Enviar comunicacoes relevantes (com seu consentimento)
  • Melhorar nossos servicos

3. Compartilhamento

Nao vendemos nem compartilhamos seus dados pessoais com terceiros, exceto para operar nossos servicos (Supabase para armazenamento, Brevo para e-mail).

4. Seus Direitos (LGPD)

Voce pode solicitar a qualquer momento:

  • Acesso aos seus dados
  • Correcao de dados incorretos
  • Exclusao dos seus dados
  • Revogacao do consentimento

5. Contato

Para exercer seus direitos ou tirar duvidas: contato@criattivados.com.br

6. Seguranca

Utilizamos criptografia SSL/TLS, Row Level Security no banco de dados e boas praticas de seguranca para proteger seus dados.
